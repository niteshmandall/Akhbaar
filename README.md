# 🚀 Akhbaar

**Daily AI & Tech Intelligence Platform + Open Dataset**

Akhbaar is a daily AI & Tech intelligence platform and an open dataset that turns raw tech reports into structured insights. It converts daily PDFs into structured JSON and a live news feed, making it easier for humans and machines to digest the fast-moving world of technology.

## 📦 Open Dataset Structure

The dataset consists of daily JSON files (e.g., `21_11_25.json`). Each file contains structured data including:

- **Title & Summary**
- **Engagement Score**
- **Categories** (AI, Robotics, Chips, Policy, Security, etc.)
- **People & Companies Involved**
- **Raw Extracted Text**
- **Metadata**
- **Unique SHA256 ID**
- **Cleaned Text** (Emojis and citations removed for better model compatibility)

This consistent structure makes it perfect for:

- 🔹 AI model training
- 🔹 Trend analysis
- 🔹 Automated content creation
- 🔹 Market intelligence
- 🔹 Research / Journalism
- 🔹 Dashboards & Apps
- 🔹 Competitor tracking

## 🤖 Why Akhbaar Matters

Most news websites show stories. Akhbaar shows **structured intelligence**.
